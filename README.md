# Catapult

SSH deployment tool for Node.js applications.

## Installation

```bash
npm install @jrmc/catapult
```

## Quick start

```bash
ctp init    # create deploy.ts
ctp deploy  # deploy
```

## Documentation

Full documentation at **https://batosai.github.io/catapult/**
